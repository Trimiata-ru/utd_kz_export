# utd_kz_export

Репозиторий хранит актуальную структуру `message.json`  
для экспорта данных по продажам из ЮТД в УТ.

---

## Правила

- Файл должен быть валидным JSON.
- Числовые значения (`price`, `sum`, `sum_nds`, `quantity` и т.д.) передаются **без кавычек**.
- Строки — в кавычках.
- Кавычки внутри строки должны быть экранированы: `\"`.
- Изменения вносятся через Pull Request.
- В одном PR — одно логическое изменение.

---

`message.json` является эталонной версией формата.
